# 学术文献获取工具

一个集成了多个学术资源平台的在线工具，帮助研究人员快速获取文献全文和引用信息。

## 功能特性

- 🔍 **APS Harvest** - 美国物理学会官方全文平台
- 📚 **Sci-Hub** - 开放获取学术文献平台
- 📖 **Library Genesis** - 数字图书馆资源
- 📝 **doi2bib** - BibTeX引用生成工具

## 使用方法

1. 输入文献的DOI号
2. 选择需要的服务平台：
   - **APS Harvest**：查看官方全文
   - **Sci-Hub**：尝试免费下载文献
   - **LibGen搜索**：在Library Genesis中搜索文献
   - **获取BibTeX**：生成引用信息
3. 系统将在新标签页中打开相应平台

## 示例DOI

- `10.1103/PhysRevC.106.064325`
- `10.1038/s41586-023-05899-8`
- `10.1126/science.abl5197`
